Abra o Console (apertar botão direito do mouse, clicar em inspecionar e ir em "Console").

Como usar:

  • let [nome da variavel] = new contiguityList(limite) - Para iniciar a lista de contiguidade. (Se ainda não foi inicializada)
  • let [nome da variavel] = new linkedList() - Para iniciar a lista encadeada. (Se ainda não foi inicializada)
  • variavel.stack(valor) - Para adicionar um novo valor à pilha.
  • variavel.remove() - Para remover o elemento do topo da pilha.
  • variavel.empty() - Para verificar se a pilha está vazia.
  • variavel.show() - Para ver o elemento do topo da pilha.
  • variavel.destroy() - Para destruir a pilha.
  • compare([lista 1].toCompare(), [lista 2].toCompare()) - Para comparar duas listas.

    • PS: as variaveis list1 e list2 já foram iniciadas.
