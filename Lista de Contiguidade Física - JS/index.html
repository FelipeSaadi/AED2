Abra o Console (apertar botão direito do mouse, clicar em inspecionar e ir em "Console")

Lista de Comandos Usáveis:

  • let [nome da variavel] = new LinkedList() - Para iniciar a lista
  • variavel.add(valor) - Para adicionar um novo nodo no final
  • variavel.printList() - Para mostrar no console a lista atual
  • variavel.remove(índice) - Para remover um nodo de um índice
  • variavel.insertAt(índice, valor) - Para adicionar um novo valor no índice desejado
  • variavel.clear() - Para limpar a lista

PS: A variável "list" já foi iniciada. (use ela ou inicie outra)
